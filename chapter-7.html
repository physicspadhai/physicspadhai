Chapter 7: Alternating Current

               

1. Alternating Current & Voltage

       

Current/voltage which changes magnitude and direction periodically with time.

       
            $$I = I_0\sin\omega t$$ or $$I = I_0\cos\omega t$$
            $$V = V_0\sin\omega t$$ or $$V = V_0\cos\omega t$$
            $I_0, V_0$ = Peak values, $\omega = 2\pi f$        
       

2. Mean and RMS Value

       
            Mean value for full cycle = 0
            Mean value for half cycle: $$I_{mean} = \frac{2I_0}{\pi} = 0.637I_0$$
            RMS value: $$I_{rms} = \frac{I_0}{\sqrt{2}} = 0.707I_0$$
            Similarly: $$V_{rms} = \frac{V_0}{\sqrt{2}}$$        
       
            Note: AC ammeter/voltmeter measure RMS values.        
       

3. AC through Resistor

       
            $$V = V_0\sin\omega t$$
            $$I = \frac{V_0}{R}\sin\omega t = I_0\sin\omega t$$
            Phase: V and I in same phase, $\phi = 0°$        
       

4. AC through Inductor

       
            $$V = V_0\sin\omega t$$
            $$I = I_0\sin(\omega t - \pi/2)$$
            Current lags voltage by $90°$
            Inductive Reactance: $$X_L = \omega L = 2\pi fL$$
            $$I_0 = \frac{V_0}{X_L}$$        
       

5. AC through Capacitor

       
            $$V = V_0\sin\omega t$$
            $$I = I_0\sin(\omega t + \pi/2)$$
            Current leads voltage by $90°$
            Capacitive Reactance: $$X_C = \frac{1}{\omega C} = \frac{1}{2\pi fC}$$
            $$I_0 = \frac{V_0}{X_C}$$        
       

6. LCR Series Circuit

       
            Impedance: $$Z = \sqrt{R^2 + (X_L - X_C)^2}$$
            $$Z = \sqrt{R^2 + (\omega L - \frac{1}{\omega C})^2}$$
            Current: $$I_0 = \frac{V_0}{Z}$$
            Phase angle: $$\tan\phi = \frac{X_L - X_C}{R} = \frac{\omega L - 1/\omega C}{R}$$        
       

7. Resonance in LCR Circuit

       

Condition when $X_L = X_C$, impedance minimum, current maximum.

       
            Resonant frequency: $$\omega_r = \frac{1}{\sqrt{LC}}$$ or $$f_r = \frac{1}{2\pi\sqrt{LC}}$$
            At resonance: $Z = R$ (minimum), $I_0 = \frac{V_0}{R}$ (maximum)
            Voltage across L = Voltage across C
            Circuit is purely resistive, $\phi = 0°$        
       

Quality Factor

       
            $$Q = \frac{\omega_r L}{R} = \frac{1}{\omega_r CR} = \frac{1}{R}\sqrt{\frac{L}{C}}$$
            Higher Q = Sharper resonance        
       

8. Power in AC Circuit

       
            Instantaneous power: $P = VI$
            Average power: $$P_{av} = V_{rms}I_{rms}\cos\phi$$
            $\cos\phi$ = Power factor
            For pure L or C: $\phi = 90°$, $P_{av} = 0$ → Wattless current
            For pure R: $\phi = 0°$, $P_{av} = V_{rms}I_{rms}$        
       

9. LC Oscillations

       
            Frequency: $$\omega = \frac{1}{\sqrt{LC}}$$
            Charge: $$q = q_0\cos\omega t$$
            Current: $$I = -I_0\sin\omega t$$
            Total energy: $$U = \frac{1}{2}\frac{q_0^2}{C} = \frac{1}{2}LI_0^2$$ = constant        
       

10. Transformer

       

Device to change AC voltage based on mutual induction.

       
            For ideal transformer: $$\frac{V_s}{V_p} = \frac{N_s}{N_p} = \frac{I_p}{I_s}$$
            Step-up: $N_s > N_p$, $V_s > V_p$, $I_s < I_p$
            Step-down: $N_s < N_p$, $V_s < V_p$, $I_s > I_p$
            Efficiency: $$\eta = \frac{P_{out}}{P_{in}} \times 100\%$$        
       
            Energy Losses: Copper loss, Iron loss, Flux leakage, Hysteresis, Eddy current loss
